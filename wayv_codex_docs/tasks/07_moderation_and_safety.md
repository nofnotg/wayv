# Task 07. Moderation & Safety

## 목표
기본 운영 체계를 구현한다.

## 범위
- report pipeline
- moderation queue
- post/comment state transitions
- admin dashboard
- high-risk gating

## 상세 작업
1. report submission
2. moderation case creation
3. admin list/detail UI
4. state change actions
5. audit logs
6. high-risk keyword rule scaffolding

## Acceptance Criteria
- 신고가 관리자 큐로 들어온다
- 게시글/댓글 상태를 변경할 수 있다
- audit log 남는다
- 제한/제거 시 사용자에게 부드럽지만 분명한 문구가 보인다
