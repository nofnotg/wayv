# Task 03. Post & Wave Reactions

## 목표
글 작성, 조회, 반응, 댓글의 기본 루프를 완성한다.

## 범위
- create post
- view post
- archive
- reaction buttons
- comments
- report entry

## 상세 작업
1. post form
2. category / emotion tag support
3. visibility scope
4. detail page
5. reactions upsert/remove
6. comments CRUD basic
7. archive action
8. report modal

## Acceptance Criteria
- 사용자가 글을 쓸 수 있다
- 반응 4종이 동작한다
- 숫자 count가 UI에 노출되지 않는다
- 댓글 작성 가능
- 아카이브/신고 동작
